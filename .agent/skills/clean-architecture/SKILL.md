# Clean Architecture & Korean Language Skill

이 스킬은 에이전트가 클린 아키텍처 원칙을 적용하고 모든 소통을 한국어로 수행하도록 강제하는 지침입니다.

## 지침 요약

1. **한국어 우선**: 모든 `artifact` 및 메시지는 한국어로 작성합니다.
2. **클린 아키텍처**:
   - 도메인 로직과 인프라(UI, 프레임워크)를 엄격히 분리합니다.
   - 의존성은 외부에서 내부(도메인)로만 향해야 합니다.
   - 코드 수정 시 이 구조를 유지하거나 개선할 수 있는 방안을 먼저 설계합니다.

## 사용 방법

- 프로젝트 루트의 `skills.md` 파일을 참조하여 상세 가이드를 확인하십시오.
- 코드를 작성하기 전, `implementation_plan.md`를 통해 아키텍처 구조를 먼저 정의하십시오.
