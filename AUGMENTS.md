# ⚡ 증강 스펙 리스트 (Augment Specs)

플레이어가 전투 승리 후 선택할 수 있는 증강(Augment) 시스템의 상세 효과 목록입니다.

## 1. 일반 등급 (Common)

| 이름 | 상세 효과 |
| :--- | :--- |
| **날카로움** | 공격력 ×1.10 |
| **축적된 분노** | 강타 공격력 ×1.20 |
| **회전 가속** | 회전베기 공격력 ×1.15 |
| **정밀 방어** | 패링 성공 시 AP +1 (턴당 1회) |
| **연속 호흡** | 전투 시작 시 AP +2 |
| **처형 본능** | 대상 HP 30% 이하일 때 공격력 ×1.15 |
| **약점 파악** | 같은 적 연속 공격 시 2타부터 공격력 ×1.10 |
| **철갑 분쇄** | 강타 적중 시 다음 턴 첫 공격 공격력 ×1.15 |
| **피의 회수** | 적 처치 시 AP +1 (턴당 2회) |
| **집중력** | 퍼펙트 패링 성공 시 다음 공격력 ×1.20 |

## 2. 에픽 등급 (Epic)

| 이름 | 상세 효과 |
| :--- | :--- |
| **약점 해부** | 같은 적 연속 공격 시 공격력 ×1.06 (최대 5중첩) |
| **과열 타격** | 강타 사용 후 다음 턴 첫 공격 공격력 ×1.35 |
| **원심 폭발** | 회전베기 공격력 ×1.25 / 처치 시 AP +1 |
| **완벽한 리듬** | 퍼펙트 패링 시 이번 턴 공격력 ×1.20 |
| **초반 가속** | 전투 시작 시 AP +3 / 첫 턴 공격력 ×1.10 |
| **피의 수확** | 적 처치 시 공격력 ×1.04 (최대 10중첩) |
| **단두의 각** | 적 HP 30% 이하일 때 강타 공격력 ×1.40 |
| **침착한 폭력** | 현재 AP 가 0~2일 때 공격력 ×1.18 |
| **관성 베기** | 기본 공격 사용 시 다음 턴 첫 공격 공격력 ×1.15 |
| **파괴의 맥박** | 매 턴 첫 공격 공격력 ×1.15 (강타라면 x1.25) |

## 3. 유니크 등급 (Unique)

| 이름 | 상세 효과 |
| :--- | :--- |
| **검성의 서약** | 모든 스킬 공격력 ×1.20 |
| **큰 한 방** | 강타 공격력 ×1.60 / 강타 처치 시 AP +2 |
| **폭풍의 중심** | 회전베기 공격력 ×1.50 / 적 2마리 이상 시 추가 x1.1 |
| **무한 템포** | 전투 시작 AP +4 / 퍼펙트 패링 시 AP +1 (턴당 2회) |
| **피의 엔진** | 적 처치 시 공격력 ×1.06 (무제한 중첩) / 피격 피해 x1.1 |
| **결투가** | 적 1마리일 때 공격력 ×1.35 / 강타 추가 x1.15 |
| **집행인의 판결** | 적 HP 40% 이하 시 공격력 x1.25 / 강타 추가 x1.2 |
| **연쇄 분쇄** | 강타 연속 사용(연속 턴) 시 공격력 ×1.50 |
| **정점의 호흡** | 턴 시작 시 AP 5 이상이면 공격력 ×1.25 |
| **완전무결** | 퍼펙트 패링 당 공격력 x1.08 (최대 3중첩) / 피격 시 소멸 |
